# NuevaVision Frontend

Frontend de sistema de gestión para óptica, desarrollado con Angular.

##  Descripción:

Aplicación web orientada a la administración de una óptica, incluyendo:

- Gestión de inventario
- Órdenes de trabajo
- Facturación
- Estadísticas
- Contabilidad
- Autenticación de usuarios

##  Tecnologías:

- Angular
- TypeScript
- SCSS

## Features destacadas:

- Arquitectura modular (layout + pages)
- Routing protegido (auth guard)
- Interceptores HTTP
- Manejo de estado básico con servicios




## Instalación local:

```bash
npm install
ng serve
